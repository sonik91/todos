    TODO

    Intro

    Todo est une aplication qui permet de lister facilement les taches à réaliser.
    Les point fort de todo sont ça simplicité a utiliser. La prise en main utilisateur est très rapide grace à ces fonctionalité simple et éfficase.
    Dans cette documentation vous retrouverez toutes les imformations utile pour utiliser notre aplication.

    Documentation utilisiateur

    Todo est une aplication qui fonctione en local. Cela implique que vous devez télécharger ce répo github sur votre terminal pour l'utiliser.
    Une fois fait vous n'avez qu'a ouvrire le fichier index.html. Ce fichier s'ouvre dans votre navigateur par default.

    Saisie d'une todo

    il suffit simplement d'ecrire le nom de ça nouvelle tache et d'apuier sur "enter" pour l'ajouter a la liste des tache a faire.

    Liste des taches

    chaque tache est representer sous cette forme. on retrouve un bouton toggle suivis du nom de la tache.
    ci-dessus est représenter une tache "active" c'est a dire qu'elle n'est pas encore noté comme réaliser.

    Tache réaliser

    Pour signier que la tache est accomplie il faut cliquer sur le bouton toggle au gauche du titre de la todo.
    dans ce cas la le titre de la todo change d'apparence et une petite coche verte aparais dans la case.

    filtre

    La liste des todos peut etre afficher sous trois format different:

    • Toutes les todos
    • Seulement les todos non-réaliser
    • Seulement les todos réaliser

    Nombre de tache qu'il réste a réaliser

    Un compteur est afficher au bas de la liste pour vous permetre de voir rapidement ce qu'il vous reste a faire.
    ce chiffre vous donne une information sur le nombre de todo qu'il vous reste a réaliser.

    Suprimer une todo

    Vous pouvez suprimer une todo directement en cliquant sur la petit croix rouge qui apparais au survole d'une todo (attention pas de retour en arrier)

    Suprimer toutes les todos réaliser

    A partir du moment ou vous avez au moins une tache annoter comme etant réaliser. Ce boutton aparais.
    Si vous cliquer dessus toutes les tache qui sont annoter comme etant réaliser seront suprimer. (Attention pas de retour en arrier)
