    # (static) _filter(force)

    Re-filtre les éléments de todo, en fonction de l'itinéraire actif. _updateCoun: Met à jour les éléments de la page, qui changent à chaque tâche terminée Si le dernier itinéraire actif n'est pas "All" ou si nous changeons d'itinéraire, nous recrée les éléments de l'élément todo, en appelant: this.show [All|Active|Completed]
    Parameters:
    Name Type Description
    force boolean | undefined force une nouvelle peinture des objets à faire.

    # (static) _updateCount()

    Met à jour les éléments de la page qui changent en fonction du nombre d'element sur la page

    # (static) _updateFilterState(currentPage)

    Met simplement à jour les états sélectionnés du navigateur de filtre * Stocker une référence à l'itinéraire actif, ce qui nous permet de re-filtrer todo éléments car ils sont marqués comme terminés ou incomplets.
    Parameters:
    Name Type Description
    currentPage obtion du filtre (all - active - completed)

    # (static) toggleComplete(id, checkbox, silent)

    met a jour le statut de l'ellement pour le faire passer d'un statut completer à actif ou l'inverse celon sont etat actuelle actualise le bouton dans le rendus ainsi que la valeur dans le stockage
    Parameters:
    Name Type Description
    id number Id de l'ellement modifier
    checkbox object la case a cocher pour verifier l'etat du statut
    silent boolean | undefined empeche le reflitrage des ellement à faire
